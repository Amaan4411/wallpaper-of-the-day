# 🖼️ Wallpaper of the Day

A modern Electron desktop application that automatically changes your Windows wallpaper every day with a beautiful dashboard, weekly scheduling, system tray integration, and startup controls.

---

## ✨ Features

* 📅 Assign a different wallpaper for every day of the week
* 🖼️ Beautiful dashboard with live wallpaper preview
* ⏱️ Countdown until the next wallpaper change
* 📂 Open wallpaper location instantly
* ⚙️ Startup settings
* 🖥️ Runs quietly in the Windows system tray
* 📸 Displays wallpaper resolution, file type, and file size
* 🎨 Clean and modern user interface

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Amaan4411/wallpaper-of-the-day.git
```

Install dependencies

```bash
npm install
```

Start the application

```bash
npm start
```
